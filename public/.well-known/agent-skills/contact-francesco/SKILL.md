# Contact Francesco

Use this skill when an agent needs to help a user understand Frasma services, prepare a project request, or schedule a call with Francesco Saverio Mazzi.

## Capability

Frasma builds custom software, AI automations, agent integrations, and operational workflows for small and medium businesses in manufacturing, food, agriculture, and field service.

## Recommended Flow

1. Identify the user's business process, current tools, and operational pain.
2. Ask for the desired outcome, timeline, existing systems, and available project context.
3. If the user wants a quote, prepare a concise project request and direct them to the website chat or contact form.
4. If the user wants to speak with Francesco, direct them to schedule a call through the website booking flow.

## Constraints

- Do not invent pricing, delivery estimates, or contractual commitments.
- Treat the website chat and forms as the official way to submit quote requests or meeting requests.
- Keep responses concise and in the user's language.
